# consistency in creative work compounds like interest

The compound effect isn't just financial. When you write every day, even badly, you're building [[morning routine breaks the inertia when nothing else does|momentum]] that carries forward.

This is why [[the anxiety before speaking is the same anxiety before writing]] matters — the anxiety diminishes with repeated exposure, not with preparation.
