# morning routine breaks the inertia when nothing else does

The hardest part of any creative day is the transition from rest to motion. A [[morning routine breaks the inertia when nothing else does|morning routine]] doesn't need to be elaborate — it just needs to be consistent enough that starting becomes automatic.

This connects to [[the anxiety before speaking is the same anxiety before writing]] because both involve overcoming an initial resistance that feels disproportionate to the actual task.

See also: [[orphan thought about unrelated things]] — contrast with how unstructured mornings lead to drift.
